# Nimbus ☁️

Nimbus is a modern glassmorphism weather app built with HTML, CSS, and JavaScript. It uses the OpenWeatherMap API to display real-time weather data for searched cities.

## Features

- Search weather by city
- Real-time weather data
- Temperature, condition, feels-like, humidity, wind, pressure, and visibility
- Loading spinner
- Success and error states
- Dynamic weather visuals
- Responsive glassmorphism UI

## Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
